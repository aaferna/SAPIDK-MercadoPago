Mediante el SDK podremos crear una cartera de Clientes para luego asociarlos a pagos o asociar tarjetas de credito o debito para cobrar pagos sin CVV (Lo que seria la nueva version de Suscripciones)

Crear un Cliente

$dataArray = array(
"accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
    "user_id" => "264334950",
    "post" => array(
        'customers' =>  array(
            'create' => true
        ),
        'data' => array(
            "email" => "agustinalejandrofernandez@mail.com.ar",
            "first_name" => "agustin",
            "last_name" => "fernandez",
            "phone" => array(
                "area_code" => "11",
                "number" => "23923157",
            ),

            "identification"=> array(
                "type" => "DNI",
                "number" => "34684457",
            ),
        ),
    ),
);

Buscar Cliente por ID

Indicando el metodo informamos el ID a buscar y obtendremos el ARRAY anterior

$dataArray = array(
"accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
"user_id" => "264334950",
    "get" => array(
        'customers' =>  array(
            "client" => true,
        ),
    'id' => "116145032-LGoQXCOsd0NLF5",
    ),
);

Listar todos los Clientes

$dataArray = array(

Eliminar Cliente

Usando el ID generado, lo indicamos para eliminar explícitamente ese usuario, nos responderá un array con los mismos datos como si una consulta o creacion se tratase

$dataArray = array(

"accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
"user_id" => "264334950",
    "delete" => array(
        'customers' =>  "116145032-LGoQXCOsd0NLF5"
    ),
);
