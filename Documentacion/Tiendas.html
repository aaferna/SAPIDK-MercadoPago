Mediante esta función podremos ver, crear y eliminar, (por ahora) las tiendas que tengamos asociadas a nuestra cuenta de Mercado Pago.

Consultar Tiendas

Esta función, muy util para saber como viene configurada nuestra cuenta, nos puede brindar los datos necesarios para la asociación de los QR's por ejemplo.

Para poder ver las Tiendas que tenemos creadas en nuestra cuenta, tendremos que ejecutar el siguiente $dataArray

"accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
"user_id" => "264334950",
    "get" => array(
        'search' => array(
            "stores" => true,
            "limit" => "30",
            "offset" => "0"
        ),
    ),
);

Mediante esta solicitud, estamos realizando la búsqueda de todas las tiendas asociadas a nuestro usuario, nos devuelve un array como este:

Tienda por ID

Mediante SAPIDK podremos obtener la información de una tienda realizando la búsqueda por su ID. Tenemos dos metodos, por MP (default y el correcto) o ML. Si usamos MP obtendremos menos información que por ML. Tan solo con indicar el metodo obtendrán la información que quieran.

$dataArray = array(
"accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
"user_id" => "264334950",
    "get" => array(
        'store' => "30349325",
        'method' => "mp" // o ml
    ),
);

Obteniendo una respuesta como la siguiente:

Crear Tiendas

Con el siguiente código podremos crear las sucursales, negocio, tiendas... Como prefieran llamarlo, mediante la API de Mercado Pago. Tengamos en cuenta que los días deben ser agregados según se nombre en ingles, detallando horarios de apertura y cierre, así como también obligatorio colocar latitude y longitude.

$dataArray = array(
"accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
"user_id" => "264334950",
    "post" => array(
        'stores' => true,
        'data' => array (
          'name' => 'Sucursal Instore',
          'business_hours' =>  array (
            'monday' => array (
                0 =>  array (
                    'open' => '08:00',
                    'close' => '12:00',
                  ),
                1 =>  array (
                    'open' => '14:00',
                    'close' => '18:00',
                  ),
            ),

            'tuesday' => array (
                0 => array (
                    'open' => '09:00',
                    'close' => '18:00',
                ),
            ),
          ),
          'location' =>  array (
            'street_number' => '3039',
            'street_name' => 'Caseros',
            'city_name' => 'Belgrano',
            'state_name' => 'Capital Federal',
            'latitude' => -32.8897322,
            'longitude' => -68.8443275,
            'reference' => '3er Piso',
          ),
          'external_id' => '1234',
        ),
    ),
);

Eliminar Tiendas

Podremos eliminar tiendas mediante la función delete que dispone SAPIDK, previamente deberemos reconocer el ID que tiene nuestra sucursal a eliminar.

Podremos eliminarla ejecutando el siguiente Array:

$dataArray = array(
    "accessToken" => "APP_USR-1864213086135942-012340-8b0cb3af04c2567f7bb8d09779aefba5-264334950",
    "user_id" => "264334950",
    "delete" => array(
        'stores' => true,
        'id' => "30301484",
        ),
    
);

Les respondera el ID de la Tienda y de Nuestro Usuario
